# 小样本学习与轻量化部署实施方案

## 1. 明确结论

采用“**自监督预训练 + 少样本微调 + 主动学习 + 蒸馏量化部署**”四步路线，在有限标注与有限算力下保持可用精度。

## 2. 小样本学习策略

### 2.1 阶段一：自监督预训练

- 数据：未标注胸片（公开数据全集）
- 方法：MoCo-v3 或 SimCLR 变体
- 目标：获得更通用的医学影像表征，降低下游标注依赖

### 2.2 阶段二：少样本微调

- 设置：每类5-shot / 10-shot / 20-shot
- 方法：参数高效微调（LoRA或冻结主干+微调分类头）
- 输出：不同shot配置下的AUC与F1曲线

### 2.3 阶段三：主动学习闭环

- 采样策略：熵最大化 + 代表性聚类混合策略
- 周期：每轮新增标注5%-10%样本
- 验证：比较随机采样 vs 主动采样的性能增益

## 3. 轻量化部署策略

### 3.1 模型压缩

- 蒸馏：教师（ConvNeXt-T全模态） -> 学生（EfficientNet-lite）
- 损失：logit蒸馏 + 中间特征蒸馏
- 目标：参数量下降 >= 50%

### 3.2 推理优化

- 导出：PyTorch -> ONNX
- 量化：FP16（GPU）与INT8（CPU）
- 引擎：ONNX Runtime / TensorRT（可选）

### 3.3 部署形态

- 云端高精度模型：科研与复核
- 边缘轻量模型：基层快速初筛
- 双模型一致性监控：当分歧大于阈值时触发复核

## 4. 性能对比模板（验收建议）

| 配置 | AUC | F1 | 延迟(ms/例) | 参数量(M) | 备注 |
|---|---:|---:|---:|---:|---|
| 全模态教师模型 | 0.89 | 0.81 | 420 | 32 | 精度上限 |
| 蒸馏学生模型 | 0.87 | 0.79 | 160 | 12 | 主部署模型 |
| INT8量化学生模型 | 0.86 | 0.78 | 95 | 12 | CPU友好 |
| 仅影像轻量基线 | 0.83 | 0.74 | 80 | 10 | 兜底方案 |

> 注：表中数值为目标区间示例，实际以实验结果为准。

## 5. 风险点与后续动作

- 风险1：量化后精度下降过大  
  后续动作：采用量化感知训练（QAT）替代纯后训练量化（PTQ）。

- 风险2：主动学习标注节奏跟不上  
  后续动作：缩短迭代周期，优先高不确定样本批次。

- 风险3：蒸馏学生模型学习不足  
  后续动作：增加特征层蒸馏与温度系数网格搜索。
