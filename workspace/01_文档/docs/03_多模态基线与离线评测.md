# 多模态基线模型与离线评测方案

## 1. 明确结论

基线原型采用“**影像编码器 + 文本编码器 + 结构化特征编码器 + 跨模态融合头**”的四段式架构，优先保障可复现，再逐步迭代性能。

## 2. 基线模型结构

## 2.1 输入定义

- 影像输入：胸片PNG/JPG，统一到`512x512`
- 文本输入：报告印象或主诉摘要（最长128 tokens）
- 结构化输入：年龄、性别、吸烟史、基础病（数值/离散）

### 2.2 编码器与融合

- 图像编码器：EfficientNet-B0（快速基线）或 ConvNeXt-T（进阶）
- 文本编码器：ClinicalBERT（英文）/ 中文医学BERT（中文语料）
- 结构化编码器：两层MLP（含LayerNorm）
- 融合方式：Cross-Attention + Gated Fusion
- 输出头：
  - 多标签分类头（BCEWithLogitsLoss）
  - 弱监督定位头（CAM分支）

## 3. 训练配置（建议）

- 优化器：AdamW，初始学习率`1e-4`
- 批大小：16（按显存调整）
- 训练轮数：30-50 epochs
- 学习率策略：Cosine Annealing
- 数据增强：随机翻转、对比度扰动、轻量仿射变换
- 类别不平衡：Focal Loss或类别重加权

## 4. 离线评测体系

### 4.1 主指标

- 分类：AUC、F1-macro、Sensitivity（高风险类）
- 定位：IoU/Pointing Game（若无像素标注则用弱监督指标）
- 解释：热图-标注重叠度（解释一致性分数）

### 4.2 分层评测

- 按年龄段/性别分层统计，检查偏差
- 按数据源（CheXpert vs MIMIC）分域测试，检查泛化
- 按模态组合（仅影像/影像+文本/全模态）做消融

### 4.3 对比基线

- Baseline-1：仅影像CNN
- Baseline-2：影像+文本拼接
- Baseline-3：全模态融合（推荐主模型）

## 5. 最小可运行原型（MVP）标准

- 可加载三类输入并输出多标签风险分数
- 可生成病灶热图（Grad-CAM）
- 可导出单例推理JSON结果（含置信度、标签、解释摘要）
- 在公开测试集达到：
  - AUC >= 0.85（MVP）
  - 高风险类敏感度 >= 0.88

## 6. 风险与后续动作

- 风险1：多模态数据缺失导致训练不稳定  
  后续动作：加入modality dropout与缺失掩码。

- 风险2：文本噪声影响融合效果  
  后续动作：先用报告“印象”字段，减少长文本噪声。

- 风险3：解释热图与医学关注点不一致  
  后续动作：引入解释一致性约束与专家抽检流程。
