# 病种范围与数据源清单（大创实施版）

## 1. 明确结论

本项目优先采用**多病种胸部异常筛查**路线，主病种设为：
- 肺炎（Pneumonia）
- 肺结节（Nodule）
- 胸腔积液（Pleural Effusion）
- 气胸（Pneumothorax）

同时保留“是否异常（Normal vs Abnormal）”二分类任务，作为轻量部署与基层初筛的兜底模式。

## 2. 关键依据与取舍

- **可扩展性**：多病种多标签任务更贴近真实临床场景，后续可增量扩展病种。
- **性能与成本平衡**：先从公开数据构建基线，再引入小样本微调，降低标注与数据获取压力。
- **可落地性**：二分类兜底模式便于早期上线验证，多标签模型用于进阶能力展示。

## 3. 数据源清单（优先级）

### A级（立项即用，公开可获取）

1. CheXpert（胸片，多标签）
   - 用途：多标签主训练集
   - 优势：规模大，标签体系成熟

2. MIMIC-CXR（胸片+报告）
   - 用途：影像-文本多模态训练
   - 优势：包含报告文本，适合跨模态融合

3. SIIM-ACR Pneumothorax（分割）
   - 用途：病灶定位/分割辅助训练
   - 优势：可增强解释模块的空间监督

### B级（条件接入）

4. LIDC-IDRI（肺结节CT）
   - 用途：CT子任务（结节检测/分类）
   - 取舍：与胸片模态差异大，建议作为扩展子课题

5. 校内合作脱敏数据（若可获得）
   - 用途：域适配与真实场景验证
   - 要求：伦理审批、脱敏协议、访问审计

## 4. 任务定义

- 主任务：胸片多标签异常识别（4病种+正常）
- 辅任务：病灶区域定位（弱监督热图，必要时引入分割子头）
- 扩展任务：CT结节子任务（可选，作为竞赛/论文增量）

## 5. 风险点与后续动作

- 风险1：公开数据标签噪声较高  
  后续动作：采用不确定样本筛查与人工复核子集。

- 风险2：多源数据分布不一致  
  后续动作：进行分域评测并引入域泛化/重加权策略。

- 风险3：校内数据接入不确定  
  后续动作：确保“仅公开数据”路径即可完成结题闭环。
